UI Controls Showcase

Here you will find different types of UI controls and examples of their usage

1. Radio Button

Radio buttons allow users to select one option from a group of mutually exclusive choices. Only one radio button in a group can be selected at a time.

Example: Choose your preferred theme

2. Checkbox

Checkboxes allow users to select multiple options from a list. Each checkbox operates independently and can be checked or unchecked without affecting others.

Example: Select your interests

3. Text Input

Text inputs allow users to enter and edit text. They can be single-line or multi-line and may include validation, placeholders, and formatting constraints.

Example: Contact form

4. Tabs

Tabs organize content into separate views that users can switch between. They help reduce clutter by showing only one section of content at a time.

Example: Product information

This is a comprehensive overview of the product features and benefits.

5. Button

Buttons trigger actions when clicked. They can be primary (main action), secondary (alternative action), or have special states like disabled or loading.

Example: Action buttons

6. Text Label

Labels provide descriptive text for form elements and help users understand what information is expected. They improve accessibility and usability.

Example: Form with labels

7. Link

Links navigate users to different pages, sections, or external resources. They can be styled as text links, buttons, or integrated into other elements.

Example: Different link types

Primary Link External Link ↗ Button-style Link

8. Tooltip

Tooltips provide additional information when users hover over or focus on an element. They help explain functionality without cluttering the interface.

Example: Hover for more info

Hover me Premium Feature ⭐

9. Dropdown List

Dropdown lists (select boxes) allow users to choose from a predefined set of options. They save space by showing options only when activated.

Example: Select your country

10. Data Grid

Data grids display structured data in rows and columns. They often include features like sorting, filtering, pagination, and inline editing for managing large datasets.
